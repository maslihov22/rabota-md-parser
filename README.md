# rabota-md-parser

Парсер вакансий с [rabota.md](https://www.rabota.md) с фильтрацией и анализом.

## Файлы

| Файл | Описание |
|------|----------|
| `parser.py` | Основной парсер. Собирает все вакансии с сайта параллельно (10 потоков), сохраняет в `vacancies_all.json`. Определяет общее количество страниц, дедуплицирует ссылки, парсит заголовок, компанию, зарплату, описание и доп. поля каждой вакансии. |
| `json_to_excel.py` | Конвертирует `vacancies_all.json` в `vacancies_all.xlsx` с форматированием: синяя шапка, автофильтры, заморозка первой строки, автоширина колонок. |
| `find_top_jobs.py` | Ищет конкретные вакансии по названию в JSON и выводит их полное описание. Список целевых вакансий задаётся вручную в коде. |
| `analyze_top10.py` | Анализирует все вакансии из Excel и выдаёт топ-10 по скоринговой системе. Учитывает: зарплату (минимум ~$1500), командировки, international-контекст, роль, рост. Автоматически исключает sales, call-center, junior, курьеров и вакансии с требованием паспорта ЕС. |

## Зависимости

```bash
pip install -r requirements.txt
```

## Запуск

```bash
# 1. Собрать все вакансии
python parser.py

# 2. Конвертировать в Excel
python json_to_excel.py

# 3. Найти топ-10 под профиль
python analyze_top10.py
```

> `find_top_jobs.py` — вспомогательный скрипт, запускается отдельно при необходимости.
